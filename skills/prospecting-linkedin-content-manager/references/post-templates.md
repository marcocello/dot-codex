# LinkedIn Post Templates

15 proven post templates. Customize with your own content, voice, and data.

---

## Template 1: The Myth Buster

```
[Common belief] is a myth.

Here's what actually happens:

[Evidence point 1]
[Evidence point 2]
[Evidence point 3]

The real truth?

[Your insight based on experience]

[Specific example or data point]

Stop believing [myth]. Start doing [alternative].

What's a myth in [your industry] that drives you crazy?
```

## Template 2: The Before/After

```
[X] years ago: [old situation]
Today: [new situation]

What changed?

Not [obvious thing].
Not [another obvious thing].

It was [unexpected insight].

Here's the full story:

[3-5 paragraphs of narrative]

The lesson that changed everything:
[One-line lesson]

What's a turning point you didn't see coming?
```

## Template 6: The Personal Failure

```

→ [Lesson 1]
```
